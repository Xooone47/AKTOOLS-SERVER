# AKTOOLS SERVER

More detail about AKTOOLS: https://github.com/akfamily/aktools?tab=readme-ov-file

## Usage

### Dev

```bash
# 如果需要创建虚拟环境
python -m venv venv

source venv/bin/activate

pip install -r requirements.txt

python -m aktools


# Exit
deactivate

```

Server running on: http://127.0.0.1:8080/
